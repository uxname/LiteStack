## Ключевые риски (требуют решения: 🔴/🟠)

1. **LOG-02 🟠 — PII в Sentry.** `setUser` отправляет email и username; Session Replay (`maskAllText: false`) пишет весь видимый текст. Это самый широкий канал утечки персональных данных.
2. **LOG-03 🟠 — потенциальная утечка чувствительных данных.** Незамаскированный replay может захватить токены/секреты из DOM; весь массив `graphQLErrors` логируется без фильтрации `extensions`.

Обе проблемы сходятся в одной точке — `replayIntegration({ maskAllText: false })` в `src/shared/lib/sentry/config.ts:19-22`. Это самое приоритетное исправление: одно изменение закрывает основной вектор и для LOG-02, и для LOG-03.
